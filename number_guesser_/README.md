# Number Guesser Game Project

## Description
This project is a game where the player has to guess a randomly generated number within a specified range. The game provides feedback on whether the guessed number is too high, too low, or correct.

## How to Run
1. Clone down this repository. and `cd` into the project directory.
2. Run 'pip install -r requirements.txt' to install the required dependencies.
3. Run the game using the command `python number_guesser.py`.
